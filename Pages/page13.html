Elemento Nulo
Segundo a propriedade do elemento nulo sempre que multiplicarmos qualquer número pelo elemento nulo, o resultado será zero! Você sabe quem é o elemento nulo? É o próprio zero. qualquer número multiplicado por zero sempre terá o produto igual a zero.
2 x 0 = 0
0 x 5 = 0
7 x 0 x 2 = 0
12.345 x 0 = 0
