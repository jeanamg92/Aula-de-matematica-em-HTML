Propriedade Associativa
Quando multiplicarmos três ou mais fatores, podemos escolher várias ordens para resolver a operação da multiplicação que o resultado será o mesmo. Vejamos de quais maneiras podemos resolver a multiplicação 3 x 5 x 7.
(3 x 5) x 7 = 15 x 7 = 105
3 x (5 x 7) = 3 x 35 = 105
5 x (3 x 7) = 5 x 21 = 105
